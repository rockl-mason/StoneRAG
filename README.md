# StoneRAG

test for sdd codex vibe coding

## Development workspace

Open the repository folder directly in VS Code. Shared project settings,
recommended extensions, and Spec Kit tasks are committed under `.vscode/`;
personal preferences such as themes, fonts, and keybindings remain user settings.

Run `Terminal: Run Task` and choose `Spec Kit: Check Environment` to verify the
local tooling. The task uses `uvx`, so [uv](https://docs.astral.sh/uv/) must be
available on the machine.
